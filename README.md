# SchulinfoApp
Die SchulinfoApp für das Gymnasium Glinde und die Sachsenwaldschule. Aktuelle Versionen sind [hier](https://github.com/Cedgetec/SchulinfoApp-android/releases)

Funktionen (abhängig von der Schule):
* Vertretungsplan
* Mensa
* News
* Klausurenplan
